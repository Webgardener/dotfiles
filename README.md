# Install

On Gnu/Linux:
```
git clone https://github.com/Webgardener/dotfiles.git
cd dotfiles
./install-dep.sh
cp -r vim ~/.vim
cp vimrc ~/.vimrc
cp tmux.conf ~/.tmux.conf
cp tmuxlinesnapshot ~/tmuxlinesnapshot
# be careful with the following lines...
cp shell/gitconfig ~/.gitconfig
cp shell/bashrc ~/.bashrc
```
